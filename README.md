# template-react-app

> React template for web app, powered by [Poi](https://poi.js.org/).

- [x] prettier + eslint
- [x] PWA

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao react-app my-project
```

### From git

```bash
sao dwarvesf/template-react-app my-project
```
